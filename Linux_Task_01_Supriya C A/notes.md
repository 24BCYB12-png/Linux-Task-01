# Linux Task 01 - Command Notes

## Navigation Commands
- `pwd` ‚Üí Print Working Directory ‚Äî shows where you are
- `ls` ‚Üí List files in current folder
- `ls -la` ‚Üí List ALL files (including hidden ones starting with `.`) with permissions
- `cd` ‚Üí Change Directory (cd alone = go home)
- `clear` ‚Üí Clears the screen
- `history` ‚Üí Shows all commands you've typed before
- `whoami` ‚Üí Shows your username
- `hostname` ‚Üí Shows your computer/machine name

## Directory Commands
- `mkdir foldername` ‚Üí Make a new folder
- `mkdir -p a/{b,c,d}` ‚Üí Make nested folders all at once
- `tree` ‚Üí Shows folder structure visually
- `cd foldername` ‚Üí Go inside a folder
- `cd ..` ‚Üí Go one folder back

## File Commands
- `touch file.txt` ‚Üí Create empty file
- `cp source destination` ‚Üí Copy file
- `mv source destination` ‚Üí Move OR rename file
- `rm file.txt` ‚Üí Delete file

## System Info Commands
- `uname -a` ‚Üí Full kernel/system info
- `hostname` ‚Üí Machine name
- `whoami` ‚Üí Current user
- `date` ‚Üí Current date and time
- `uptime` ‚Üí How long system has been running
- `pwd` ‚Üí Current directory path

## Important Tips
- Linux is CASE SENSITIVE ‚Üí `Reports` ‚âÝ `reports`
- Always check your location with `pwd` before running file commands
- Use `ls` to confirm files/folders exist before copying/moving
- `mkdir Reports` is different from the folder created as `Rports` ‚Äî typos matter!
